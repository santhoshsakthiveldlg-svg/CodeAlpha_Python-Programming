# codealpha_tasks

🚀 CodeAlpha Python Programming Tasks

📌 Overview

This repository contains multiple beginner-friendly Python projects developed as part of the CodeAlpha Python Programming Internship.
Each task focuses on strengthening core Python concepts such as loops, conditionals, functions, dictionaries, and user input/output.

All programs are console-based and require no external libraries, making them ideal for beginners.


📂 Projects Included

🕹 Task 1: Hangman Game

📈 Task 2: Stock Portfolio Tracker

💬 Task 4: Basic Chatbot

🛠 Requirements

* Python 3.x
* Any Code Editor (VS Code recommended)
* Command Prompt / Terminal

▶️ How to Run the Programs

1. Open Terminal / Command Prompt
2. Navigate to the project folder
3. Run the required Python file:

   * python task1_hangman.py
   * python task2_stock_portfolio.py
   * python task4_chatbot.py

🧩 Task Details

✅ Task 1: Hangman Game 🕹

Description:

A text-based Hangman game where the player guesses letters to reveal a hidden word within a limited number of attempts.

Features:

 * Random word selection
 * Letter-by-letter guessing
 * Limited incorrect attempts
 * Win/Lose messages

Concepts Used:

* random module
* while loop
* if-else conditions
* Strings & Lists
* User input/output


✅ Task 2: Stock Portfolio Tracker 📈

Description:

  A simple stock tracking program that calculates total investment based on predefined stock prices and user input.

Features:

  * Predefined stock price dictionary
  * User input for stock name and quantity
  * Automatic investment calculation
  * Portfolio summary display

Concepts Used:

 * Dictionaries
 * Loops
 * Arithmetic operations
 * Input/Output
 * Basic error handling


✅ Task 4: Basic Chatbot 💬

Description:

  A rule-based chatbot that responds to predefined user inputs.

Features:

  * Responds to greetings
  * Handles exit commands
  * Simple conversational flow

Example Inputs:

  * hello
  * how are you
  * bye

Concepts Used:

  * if-elif-else conditions
  * Functions
  * Loops
  * String handling
  * Input/Output
